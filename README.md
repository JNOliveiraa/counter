 Contador em JavaScript

Projeto de contador desenvolvido utilizando JavaScript puro, com manipulação de DOM e persistência de dados no navegador.

 Funcionalidades

 Incrementar número
 Decrementar número
 Resetar contador
 Auto incremento
 Auto decremento
 Salvamento automático com LocalStorage
 Mudança de cor quando o número fica negativo
